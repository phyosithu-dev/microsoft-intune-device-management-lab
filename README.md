# Microsoft Intune Device Management Lab

## Overview

This project demonstrates hands-on experience with Microsoft Intune and Microsoft Entra ID in a Microsoft 365 Business Premium environment.

## Technologies Used

* Microsoft Intune
* Microsoft Entra ID
* Microsoft 365 Business Premium
* Windows 11
* Security Groups
* Compliance Policies
* Configuration Profiles

## Lab Tasks Completed

### Device Enrollment

* Configured MDM user scope.
* Enrolled Windows 11 device (INTUNE-CLIENT01) into Microsoft Intune.
* Verified successful device registration and management.

### Security Groups

* Created a Security Group for Intune policy assignments.
* Added users to the Security Group.
* Assigned policies using group-based management.

### Compliance Policies

* Created Windows compliance policies.
* Configured password and security requirements.
* Verified device compliance status.

### Configuration Profiles

* Created a Windows Device Lock configuration profile.
* Configured automatic device lock after inactivity.
* Assigned profile through Security Group targeting.
* Verified successful policy deployment.

* Microsoft Entra ID Administration
